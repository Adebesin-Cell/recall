# Recall

Quick brain games that push your memory, focus, and reaction — playable in seconds,
shareable with a link. No sign-up, no app to install.

> Built on top of **Panda CSS v2 (beta)**.

## The games

- **Numbers** — a number flashes; memorize it, then type it back before the timer
  drains. Every level adds a digit and less time.
- **Stroop** — a color word appears painted in some ink color. Tap **MATCH** or
  **NO MATCH** before the clock runs out. It keeps speeding up.
- **Math Sprint** and **Word Form** — coming soon.

## How to play

1. Open the app and pick a game.
2. Hit **Start** and survive as long as you can — one miss (or a timeout) ends the run.
3. Each level is faster and harder than the last. Your best score is saved on your device.

Tip: tap the **?** icon in any game for a quick rules refresher.

## Challenge your friends

When a run ends you get a **leaderboard** — and a link.

- Set your name once in **Settings** (the gear, top-right).
- Share your link. Whoever opens it plays the **exact same sequence** you did, so
  scores are truly comparable.
- Their result joins the board, and they can pass the link on. The leaderboard rides
  along in the link itself — there's no server keeping score.
- Play again and your entry updates (it keeps your best and counts how many times
  you've played).

## Running it locally

```bash
bun install
bun run dev
```

Then open the local URL it prints. That's it — have fun. 🟣
